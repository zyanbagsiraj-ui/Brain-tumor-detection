About This Project

This project aims to develop a Convolutional Neural Network (CNN) model capable of detecting brain tumors from MRI scans, focusing on tumor types such as Glioma, Meningioma, Pituitary, and No-Tumor categories.

The system enhances medical diagnosis by providing fast and accurate tumor classification using deep learning technology.
